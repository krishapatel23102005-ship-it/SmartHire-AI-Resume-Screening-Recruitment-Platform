About ResuAI

AI Recruitment & Career Recommendation Platform

ResuAI is an intelligence-driven recruitment matching portal designed to bridge the gap between candidate qualifications and corporate hiring criteria. By employing text extraction and pattern matching algorithms, we help job seekers format their profiles to bypass automated screening filters while offering recruiters ranked talent matching dashboards.

Our Matching Methodology

Unlike standard keyword searching, ResuAI calculates match weights dynamically. Our system scans the candidate's PDF profile, indexes their capabilities, and calculates a match percentage against an employer's requirements: 

Match Score = (Matched Skills / Required Skills) * 100

Key Technologies

  • Backend: Django Model-View-Template framework
  • Database: MySQL relational database storage with custom startup schema validators
  • Extraction: Native PDF byte extraction engine using pypdf
  • UI/UX: Glassmorphic CSS design system with dynamically generated SVG line graphs
